USB protocol from device perspective

Aug 23, 2024

Embedded Assembly

Purpose:

The goal of this report is for me to learn and develop my own USB device stack. I've got moderate experience working within embedded enviorments as I have been working on my own hardware abstraction layer for the Pico. However the USB protocol has been especially difficult for me, partly because there is a complete lack of good documentation or helpful material online, so maybe this could be of some help to some of you.

Method:

I'll be making baremetal firmware for my Rasberry Pi Pico that will: 1. sucessfully show up as a device on my host pc; accomplishing the USB setup sequence and 2. in device mode emulate and function as a regular keyboard device.
